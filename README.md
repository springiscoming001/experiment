# Tree Layout Alignment Strategy User Experiment

## 实验简介

本实验旨在评估不同**树节点对齐策略**（Alignment Strategy）对用户寻找**最近公共祖先(LCA)**任务的影响。

## 什么是 LCA？

**最近公共祖先（Lowest Common Ancestor，LCA）** 是指两个或多个节点在树中**最早汇合的共同祖先**。

例如在下图中，节点 4 和节点 5 的 LCA 是节点 2：
```
       1
      / \
     2   3
    / \
   4   5
```

## 实验流程

### 1. 练习阶段（3个任务）
- 帮助您熟悉实验操作和界面
- 每次选择后会显示正确答案

### 2. 正式实验（96个任务）
- **24棵树** × **4种对齐策略** = 96个任务
- 相同树结构的任务至少间隔3个位置
- **4个**红色标记的叶子节点

## 对齐策略说明

| 策略 | 描述 |
|------|------|
| midpoint | 子节点位置的中点 |
| midZ | 最大子树的中间位置 |
| midDepthZ | 最深子树的中间位置 |
| weightedSquared | 最小化加权平方距离 |

## 操作说明

1. 点击 **"Start Experiment"** 开始实验
2. 找到红色圆圈标记的节点（共4个）
3. 点击这些节点的**最近公共祖先**
4. 练习阶段结束后自动进入正式实验
5. 实验结束后请导出结果（CSV格式）
